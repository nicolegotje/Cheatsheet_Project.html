HTML Cheatsheets



HTML Tags

Tag Name Description
<a> Hyperlink element Defines a hyperlink.
<article> Article element Defines an article.
<audio> Audio element Defines embedded sound content.
<b> Bold element Defines bold text.
<blockquote> Blockquote element Defines a section that is quoted from another source.
<body> Body element Defines the document's body.
<div> Content Division element The generic container for flow content.
<em> Emphasized element Defines emphasized text.
<embed> Embeded element Defines a container for an external application.
<footer> Footer element Defines a footer for a document or section.
<form> Form element Defines an HTML form for user input.
<p> Paragraph element HTML element represents a paragraph.


Table Tags

Tag Name Description
<table> Table element The wrapper element for all HTML tables.
<thead> Table header element The set of rows defining the column headers in a table.
<tbody> Table body element The set of rows containing actual table data.
<tr> Table row element The table row container.
<td> Table data element Table data elements are the data containers of the table.
<tfoot> Table footer element The set of rows defining the footer in a table.
